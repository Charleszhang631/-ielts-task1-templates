# 🧾 IELTS Task 1 Writing Templates | 雅思小作文高分写作模版

本仓库系统整理了雅思写作 Task 1（Academic）各类图表题型的**高分模版句框**，包括引言段（Introduction）、概览段（Overview）、主体段（Body）等部分，帮助考生迅速掌握各类图表题的结构化写作方式。

---

## 📌 仓库亮点 Features

- ✅ **覆盖所有7类图表题型**：柱状图、折线图、饼图、表格、流程图、地图题、混合图
- ✅ **按段落模块整理模版句型**：Intro / Overview / Body1 / Body2
- ✅ **中英双语对照模版**：每个模版配有常见句型+中文解释+高分示例
- ✅ **实用写作建议**：每个模版附带段落写作技巧与注意事项

## 🧭 使用建议 How to Use

1. 根据题目题型，选择相应的 `.md` 文件查阅写作结构。
2. 熟悉各段常用句式，并结合真实题目进行套用或仿写。
3. 练习使用连接词、比较结构、被动语态等常用表达。
4. 搭配本系列其他词汇句型仓库（如 `ielts-task1-verbs-collection`）使用，效果更佳。
